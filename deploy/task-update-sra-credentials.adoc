= 更新儲存複寫介面卡（SRA）認證
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
若要讓SRM與SRA通訊、如果您已修改認證資料、則應更新SRM伺服器上的SRA認證資料。

您應該已經執行「在SRM應用裝置上設定SRA」主題中所述的步驟。

xref:task-configure-sra-on-srm-appliance.adoc[在SRM設備上設定SRA]

.步驟
. 刪除'/SRM/SRA/conf'Directory使用：
+
.. CD /SRM/SRA/conf
.. RM -RF *


. 執行perl命令以新認證設定SRA：
+
.. CD /SRM/SRA/
.. Perl command.pl -I <VA-IP>管理員<VA-password>
